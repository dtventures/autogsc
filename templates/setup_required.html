OAuth Setup Required

To enable Google Sign-In, you need to create OAuth credentials:

  1. Go to Google Cloud Console
  2. Click "Create Credentials" → "OAuth client ID"
  3. Choose "Web application"
  4. Add Authorized redirect URI: http://localhost:5000/oauth/callback
  5. Click "Create" and download the JSON
  6. Save as client_secret.json in the AutoGSC folder
  7. Restart the app
